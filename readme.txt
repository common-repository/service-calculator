== Description ==

Service calculator is simple plugin that allows your site visitors to calculate your services prices.


== Installation ==

1. Install either via the WordPress.org plugin directory, or by uploading the files to your server.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Get to settings -> service calculator
1. Create table of services
1. Add shortcode to your page

== Frequently Asked Questions ==

= What is the shortcode for this plugin =

Shortcode for this plugin is [service_calculator]
